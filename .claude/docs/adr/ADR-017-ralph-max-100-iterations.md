# ADR-017: ralph max iteration 100 설정

- **상태**: Accepted
- **날짜**: 2026-03-19

## 결정
ralph 루프의 최대 반복 횟수를 **100**으로 설정 (AW-007).

## 이유
- 복잡한 구현은 1-2회로 완료 불가 → 충분한 반복 허용
- 무한 루프 방지 (100회 상한)
- 3+ 반복에서 동일 오류 → 사용자에게 근본 문제 보고 (판단 기준)

## Gotchas
- 100회 = 비용이 클 수 있음 → 빠른 단순 수정은 `--no-prd` 사용
- 동일 오류 반복 3회 → 중단하고 사용자 보고 필요
