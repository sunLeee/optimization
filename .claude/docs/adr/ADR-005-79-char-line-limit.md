# ADR-005: Python 줄 길이 79자 제한

- **상태**: Accepted
- **날짜**: 2026-03-19

## 맥락
Black(88자)과 PEP8(79자) 사이에서 팀 기준 필요. 내부에서 세로로 분할 화면 사용 시 88자도 긴 경우 발생.

## 결정
**79자** (PEP8 권장값)로 고정.

## 이유
- 세로 분할 화면에서 여백 확보
- PEP8 원래 권장값 준수
- 터미널 80자 기준과 일치

## 대안 검토
| 대안 | 거부 이유 |
|------|---------|
| Black 88자 | 분할 화면에서 줄 바꿈 발생 |
| 제한 없음 | 수평 스크롤 필요 |

## 결과
- pyproject.toml `line-length = 79` 설정
- `convention-python` skill의 핵심 Gotchas 항목
