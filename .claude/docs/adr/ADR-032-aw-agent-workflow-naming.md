# ADR-032: AW (Agent Workflow) 규칙 번호 체계

- **상태**: Accepted
- **날짜**: 2026-03-19

## 결정
AW = Agent Workflow. AW-001~045 범위로 에이전트 운영 규칙 번호 부여.

## 이유
- 규칙에 번호 → CLAUDE.md에서 "AW-007"로 간단히 참조
- 팀원이 "AW-005 위반"으로 명확히 커뮤니케이션 가능
- 의사결정 추적 용이

## 관리 위치
- `.claude/docs/team-operations.md` — 전체 목록
- `CLAUDE.md` § Team Operations — 그룹별 요약

## 정의
AW = Agent Workflow (에이전트가 작업할 때 따라야 하는 워크플로우 규칙)
